# 🩸 LifeLink – Smart Blood Donation Network

LifeLink is a full-stack web application that connects blood donors and recipients in real time. The platform enables efficient donor discovery and emergency request handling through a simple and user-driven interface.

---

## 🚀 Features

* 🔍 Smart donor search (blood group, location, availability)
* 📍 Location-based filtering
* ⚡ Emergency blood request system
* 📊 Dashboard with basic insights
* 👤 Donor profile management
* 🌙 Dark / Light mode support
* 💬 Basic communication between users

---

## ⚠️ Note

This project is developed for demonstration and portfolio purposes.

* No preloaded or sample data is included
* Users must manually register and create data
* Data is stored locally using H2 and resets on server restart

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite, Tailwind CSS
* **Backend:** Spring Boot (REST API)
* **Database:** H2 (Local Development)
* **Authentication:** JWT-based authentication

---

## ▶️ How to Run

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 💡 Usage

1. Register or login to the system
2. Create a donor profile
3. Search for donors using filters
4. Raise blood requests when needed
